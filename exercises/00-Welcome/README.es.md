# `000` Bienvenido a Master TypeScript: Objetos

Este tutorial está integrado con LearnPack y evaluación automática.

## Objetivo de este tutorial

En este LearnPack vas a practicar TypeScript resolviendo ejercicios progresivos sobre objetos tipados, interfaces y datos estructurados.

## Flujo recomendado

1. Lee las instrucciones del ejercicio.
2. Implementa tu solución en `app.ts`.
3. Haz clic en **Run** para revisar la salida.
4. Haz clic en **Test** y repite hasta pasar todos los tests.

## Enfoque de este LearnPack

- **Rango:** `001-055`
- **Enfoque:** interfaces, propiedades opcionales, datos anidados, mutación vs inmutabilidad y paso por valor o referencia
- **Dificultad:** intermedio

Cuando estés listo, haz clic en **Next** arriba a la derecha.
