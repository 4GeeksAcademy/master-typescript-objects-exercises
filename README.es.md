## Instalación en un clic

Abre en [Codespaces](https://codespaces.new/?repo=4GeeksAcademy/master-typescript-objects-exercises) o [Gitpod](https://gitpod.io#https://github.com/4GeeksAcademy/master-typescript-objects-exercises).

## Instalación local

1. Instala LearnPack:

```bash
npm i @learnpack/learnpack -g
```

2. Instala el plugin local de TypeScript para LearnPack:

```bash
learnpack plugins:install file:$(pwd)/learnpack-typescript-plugin
```

3. Inicia los ejercicios:

```bash
learnpack start
```

## Contrato de archivos por ejercicio

Cada ejercicio contiene:

1. `app.ts`: archivo principal oficial del estudiante (solo base inicial).
2. `app.js`: entry runtime para el botón `Run` de LearnPack.
3. `README.md` + `README.es.md`: instrucciones.
4. `test.js`: script de evaluación automática.

`app.ts` y `app.js` se mantienen intencionalmente como plantillas iniciales (sin lógica resuelta).
